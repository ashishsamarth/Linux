To list only links  :-  ls -lrt|grep "^l"
To list only directories    :-  ls -lrt|grep "^d"
To list only files  :-  ls -lrt|grep "^-"
To list the last EDITED / MODIFIED file :-  ls -lt
To list the last EDITED / MODIFIED file in reverse order    :-  ls -ltr
To list the file in human readable size :-  ls -lh
To list the directory info without showing its contents and without entering the directory  :-  ls -ldh Directory_Name
To list hidden files in current directory   :-  ls -la
To list hidden files in current directory and Parent Directory  :-  ls -lA
